# Mimi Tracker

VALORANTの戦績を詳細に表示する非公式の戦績トラッカーです。

## プロジェクト概要

Mimi Trackerは、Riot Games APIを使用してVALORANTプレイヤーの戦績情報を詳細に分析・表示するWebアプリケーションです。Tracker.ggのような高級感のあるデザインで、詳細な統計情報を提供します。

## 主な機能

- **RSO認証**: Riot Sign Onによる安全なログイン機能
- **戦績表示**: ログインしたユーザーの詳細な戦績情報を表示
- **詳細統計**: K/D、ADR、KAST、DDΔ、HS率などの詳細な統計情報
- **マッチ履歴**: 最近のマッチの詳細な情報を表示
- **エージェント分析**: 最も使用しているエージェントTOP3の統計
- **Tracker Score**: パフォーマンス、一貫性、影響力を数値化
- **味方運分析**: チームメイトの実力を分析
- **ショップ情報**: 今日のショップアイテムを表示

## 技術スタック

- **フロントエンド**: HTML5, CSS3, JavaScript
- **API**: Riot Games API (VALORANT API)
- **バックエンド**: Python（実装予定）

## 認証について

VALORANT APIの仕様により、個人の戦績データ（マッチ履歴、統計情報など）を取得するには、RSO（Riot Sign On）認証が必須です。

- ユーザーは自分のRiotアカウントでログインする必要があります
- ユーザーが戦績データの共有に明示的に同意する必要があります
- プロフィールの公開/非公開設定を実装予定です

## 使用するAPIエンドポイント

### RSO認証必須のエンドポイント
- `/val/match/v1/matches/{puuid}` - マッチ履歴取得（RSO認証必須）
- `/val/match/v1/matchlists/by-puuid/{puuid}` - マッチリスト取得（RSO認証必須）
- `/val/ranked/v1/leaderboards/by-act/{actId}` - ランキング取得

### 公開APIエンドポイント
- `/riot/account/v1/accounts/by-riot-id/{gameName}/{tagLine}` - PUUID取得
- `/val/content/v1/contents` - コンテンツ情報取得（エージェント、マップ、武器情報など）

## ユーザー体験の流れ

1. **ログイン**: RSO（Riot Sign On）認証により、ユーザーが自分のRiotアカウントでログイン
2. **同意取得**: ユーザーが戦績データの共有に明示的に同意
3. **戦績表示**: ログインしたユーザーの詳細な戦績情報が表示される
4. **統計確認**: 統計情報、マッチ履歴、エージェント分析などを確認
5. **公開設定**: ユーザーが自分のプロフィールを公開/非公開に設定可能

## プライバシーとセキュリティ

- プライバシーポリシーと利用規約をサイトに明記
- RSO（Riot Sign On）認証を必須として実装
- ユーザーデータはRiot Games APIから直接取得し、サーバーに永続保存しない
- プロフィールの公開/非公開設定を実装し、ユーザーが自分のデータの公開範囲を制御できるようにします
